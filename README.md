# Dad Jokes Web App

This is a simple web application that fetches random dad jokes from the [icanhazdadjoke](https://icanhazdadjoke.com/) API.

## Link
https://dad-jokes-web-app.vercel.app/

## Features

- Fetches a random dad joke on load
- Includes a button to fetch a new joke on demand
- Clean and minimal UI

## Tech Stack

- HTML, CSS, JavaScript
- [icanhazdadjoke API](https://icanhazdadjoke.com/api)

## How to Run

1. Clone or download the repository
2. Open `index.html` in your browser
3. Enjoy the jokes!

## Deployment

You can host this on GitHub Pages or Vercel by using `index.html` as the entry point.
